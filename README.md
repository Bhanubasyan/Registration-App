# Registration App

This is a simple registration form web application built using HTML, CSS, and JavaScript.  
It allows users to register with their details and stores the data (if connected with a backend).

## Features
- User-friendly registration form
- Input validation
- Simple styling with CSS

## Project Structure
- `registration.html` - Main registration page
- `style.css` - Styling for the app
- `scripts.js` - JavaScript for form handling
- `server.js` - Backend server file
